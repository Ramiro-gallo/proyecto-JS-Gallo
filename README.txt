Hola! Espero que estés teniendo un buen día. 

Bienvenido/a a mi calculadora de calorías, BMR y TDEE.

Este proyecto forma parte de mi instrucción como programador, puntualmente del curso de 
JavaScript de Coderhouse.

Consiste en una calculadora de calorías completamente funcional que ofrece la posibilidad
de seleccionar la opción de fórmula de cálculo más adecuada. Las opciones incluyen:

Harris-Benedict Equation
men: BMR = (13.75 × weight [kg]) + (5.003 × height [cm]) – (6.775 × age [years]) + 66.5
women: BMR = (9.563 × weight [kg]) + (1.850 × height [cm]) – (4.676 × age [years]) + 655.1

Revised Harris-Benedict Equation
men: BMR = (13.397 × weight [kg]) + (4.799 × height [cm]) – ( 5.677 × age [years]) + 88.362
women: BMR = (9.247 × weight [kg]) + (3.098 × height [cm]) – (4.330 × age [years]) + 447.593

Mifflin St Jeor Equation
men: BMR = (10 × weight [kg]) + (6.25 × height [cm]) – (5 × age [years]) + 5
women: BMR = (10 × weight [kg]) + (6.25 × height [cm]) – (5 × age [years]) – 161

El foco del proyecto está puesto en cumplir al pie de la letra con las consignas requeridas,
procurando utilizar las mejores prácticas de código limpio y escalabilidad.
